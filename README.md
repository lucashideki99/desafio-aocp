# 📊 Sistema de Controle de Contas

- Desenvolvedor: Lucas Hideki Yamanaka  
- Sistema web para gerenciamento de contas bancárias, desenvolvido com Java Web (JSP/Servlets), JDBC e PostgreSQL. O projeto permite cadastro, consulta, atualização, ativação/desativação e controle de saldo de contas.

---

## 🚀 Tecnologias Utilizadas

- Java 11  
- JSP  
- Servlets  
- Apache Tomcat 9  
- PostgreSQL  
- Docker  
- JDBC  
- NetBeans IDE  
- Maven  

---

## ▶️ Execução do Projeto

### 1. Subir o banco de dados

para subir o banco de dados pegue o docker-compose.yaml execute o seguinte comando no terminal: 

docker-compose up -d

Isso irá iniciar o PostgreSQL automaticamente.

---

### 2. Criar as tabelas no banco de dados

Após subir o container, execute o script de criação das tabelas disponível no projeto, junto com algums dados iniciais.


---

### 3. Importe o projeto na IDE 

A IDE utilizada no prjeto foi Netbeans 21

Após a importação faça um clean e build, para baixar as bibliotecas do projeto e execute o projeto.
